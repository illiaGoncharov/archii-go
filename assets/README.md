# Assets

Эта папка содержит изображения, иконки и другие статические ресурсы приложения.

## Необходимые файлы

Для работы приложения нужны следующие файлы:

### icon.png
- Размер: 1024x1024px
- Формат: PNG
- Иконка приложения

### splash.png
- Размер: 1284x2778px (для iPhone)
- Формат: PNG
- Splash screen (загрузочный экран)

### adaptive-icon.png
- Размер: 1024x1024px
- Формат: PNG
- Adaptive icon для Android

### favicon.png
- Размер: 48x48px
- Формат: PNG
- Favicon для веб-версии

## Временные placeholder'ы

Пока можно использовать простые placeholder изображения. Позже замени их на настоящий дизайн.

Для быстрого создания placeholder'ов можно использовать:
- [Figma](https://figma.com)
- [Canva](https://canva.com)
- Или любой графический редактор

## Дизайн

Цветовая схема приложения:
- Основной цвет: `#3B82F6` (синий)
- Акцент: `#F59E0B` (золотой для legendary элементов)
- Фон: `#F9FAFB` (светло-серый)
- Текст: `#111827` (тёмно-серый)
