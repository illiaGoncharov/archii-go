## Обзор программы

### Цели партнёрской программы

1. **Привлечение пользователей** — партнёры приводят новую аудиторию
2. **Создание контента** — экспертный контент от профессионалов
3. **Повышение доверия** — ассоциация с известными именами
4. **Масштабирование** — рост без пропорционального увеличения затрат

### Типы партнёров

| Тип | Описание | Основная ценность | Вознаграждение |
|-----|----------|-------------------|----------------|
| Гиды | Профессиональные экскурсоводы | Авторские маршруты | Revenue share |
| Краеведы | Историки, исследователи | Экспертный контент | Роялти |
| Блогеры | Инфлюенсеры, авторы каналов | Охват аудитории | Комиссия |
| Хранители | Активные волонтёры | UGC, модерация | Бонусы в приложении |
| Организации | Музеи, школы, НКО | Партнёрские проекты | Индивидуально |

### Принципы программы

```
1. ПРОЗРАЧНОСТЬ
   Чёткие условия, понятный расчёт вознаграждения

2. СПРАВЕДЛИВОСТЬ
   Вознаграждение пропорционально вкладу

3. КАЧЕСТВО
   Все материалы проходят модерацию

4. ДОЛГОСРОЧНОСТЬ
   Фокус на устойчивых отношениях, не разовых акциях

5. ПОДДЕРЖКА
   Помогаем партнёрам создавать качественный контент
```

## Программа для гидов

### «Официальный гид Archi-Go»

#### Кто может стать

- Профессиональные экскурсоводы с опытом от 1 года
- Авторы архитектурных прогулок
- Гиды музеев и культурных центров
- Преподаватели истории архитектуры

#### Что получает гид

| Преимущество | Описание |
|--------------|----------|
| **Авторский профиль** | Страница в приложении с фото, био, рейтингом |
| **Создание маршрутов** | Бесплатный Pro-аккаунт для создания маршрутов |
| **Монетизация** | 70% от продажи маршрутов |
| **Продвижение** | Реклама в приложении и соцсетях |
| **Бейдж** | Официальный бейдж «Archi-Go Guide» |
| **Аналитика** | Данные о прохождениях маршрутов |

#### Модель монетизации для гидов

```
ВАРИАНТ A: Платные маршруты

Гид устанавливает цену маршрута: 99-499 ₽
Распределение:
• Гиду: 70%
• Archi-Go: 30% (комиссия + процессинг)

Пример:
• Маршрут «Модерн на Патриарших» — 199 ₽
• 100 покупок в месяц = 19 900 ₽
• Гиду: 13 930 ₽
• Archi-Go: 5 970 ₽

ВАРИАНТ B: Эксклюзивные маршруты в подписке

Гид создаёт маршруты, доступные только Premium-подписчикам
Распределение:
• Фиксированная плата: 5 000 ₽ за маршрут
• + 5% от подписок пользователей, прошедших маршрут

ВАРИАНТ C: Комбинированная модель

• Базовые маршруты — бесплатно (для охвата)
• Продвинутые маршруты — платно (для монетизации)
```

#### Процесс сертификации

```
ШАГ 1: ЗАЯВКА
• Заполнить форму на сайте
• Приложить портфолио (описание 2-3 экскурсий)
• Указать опыт и специализацию

ШАГ 2: ПРОВЕРКА
• Верификация опыта (отзывы, публикации)
• Тестовое задание: описание 1 маршрута
• Интервью (опционально)

ШАГ 3: ОБУЧЕНИЕ
• Онлайн-курс «Создание маршрутов в Archi-Go» (2 часа)
• Гайд по стилю и требованиям к контенту
• Технический onboarding

ШАГ 4: АКТИВАЦИЯ
• Создание профиля
• Публикация первого маршрута (после модерации)
• Получение бейджа «Archi-Go Guide»

Срок: 1-2 недели
```

#### Требования к маршрутам

```
СТРУКТУРА МАРШРУТА:

1. Название (до 50 символов)
   ✓ «Модерн на Патриарших»
   ✗ «Прогулка по району»

2. Описание (200-500 символов)
   • Что увидим
   • Для кого подходит
   • Уровень сложности

3. Точки маршрута (5-15 точек)
   • Название точки
   • Координаты
   • Описание (100-300 символов)
   • Какие элементы искать
   • Фото референс

4. Метаданные
   • Продолжительность (1-3 часа)
   • Расстояние (1-5 км)
   • Сложность (1-5)
   • Доступность (коляски, пожилые)

КАЧЕСТВО:

• Грамотный русский язык
• Проверенные факты
• Актуальные координаты
• Оригинальный контент (не копипаст)
```

#### Договор с гидом (ключевые пункты)

```
1. ПРЕДМЕТ
   Гид создаёт авторские маршруты для размещения в Archi-Go

2. ПРАВА
   • Гид сохраняет авторские права на контент
   • Archi-Go получает неэксклюзивную лицензию на использование
   • Гид может удалить маршрут с уведомлением за 30 дней

3. ВОЗНАГРАЖДЕНИЕ
   • 70% от продаж маршрутов
   • Выплаты ежемесячно при сумме от 1000 ₽
   • На карту или ЮMoney

4. ОБЯЗАТЕЛЬСТВА ГИДА
   • Создавать качественный оригинальный контент
   • Поддерживать актуальность информации
   • Не нарушать права третьих лиц

5. ОБЯЗАТЕЛЬСТВА ARCHI-GO
   • Размещать маршруты в приложении
   • Своевременно выплачивать вознаграждение
   • Предоставлять аналитику
```

| **Доступ к материалам** | Ранний доступ к UGC для верификации |

#### Модель вознаграждения

```
ОПЛАТА ЗА КОНТЕНТ:

Тип контента                    | Ставка

Описание элемента (краткое)     | 500 ₽

Верификация UGC (за элемент)    | 100 ₽

РОЯЛТИ:

Дополнительно 10% от продаж маршрутов, 
где использован контент эксперта.

УСЛОВИЯ:
```

```
ШАГ 3: РЕГУЛЯРНАЯ РАБОТА

• Создание контента
• Редактура и согласование
• Публикация и оплата

ШАГ 4: ЭКСПЕРТНЫЙ СТАТУС
```

## Программа «Хранители»

### Волонтёрская программа

#### Суть

«Хранители» — активные пользователи, которые помогают:
- Верифицировать находки других пользователей
- Добавлять новые элементы в базу
- Модерировать контент
- Помогать новичкам

#### Уровни Хранителей

```
УРОВЕНЬ 1: НАБЛЮДАТЕЛЬ
────────────────────────
Требования:
• 30+ найденных элементов
• 10+ дней активности
• Аккаунт старше 1 месяца

Возможности:
• Голосовать за/против находок
• Предлагать исправления описаний

Награды:
• Бейдж «Наблюдатель»
• +10% очков за находки


УРОВЕНЬ 2: ХРАНИТЕЛЬ
────────────────────────
Требования:
• 100+ найденных элементов
• 50+ одобренных голосований
• Пройти тест на знание элементов

Возможности:
• Верификация находок (решающий голос)
• Добавление новых POI
• Редактирование описаний

Награды:
• Бейдж «Хранитель»
• Premium на 3 месяца
• Приглашение в закрытый чат


УРОВЕНЬ 3: СТАРШИЙ ХРАНИТЕЛЬ
────────────────────────────────
Требования:
• 300+ найденных элементов
• 200+ верификаций
• Рекомендация от другого Старшего Хранителя

Возможности:
• Добавление новых элементов в базу
• Модерация споров
• Участие в разработке гайдлайнов

Награды:
• Бейдж «Старший Хранитель»
• Пожизненный Premium
• Мерч Archi-Go
• Приглашения на все мероприятия
```

#### Система мотивации

```
ОЧКИ ХРАНИТЕЛЯ:

Действие                        | Очки
────────────────────────────────|──────
Верификация находки             | 5
Предложение исправления         | 10
Добавление POI (одобрено)       | 50
Добавление элемента (одобрено)  | 100
Помощь новичку в чате           | 10

ОБМЕН ОЧКОВ:

• 500 очков = 1 месяц Premium
• 1000 очков = эксклюзивный элемент
• 2000 очков = мерч (футболка)
• 5000 очков = встреча с командой
```

### API для партнёров

```typescript
// Эндпоинты для гидов и партнёров

{
  "title": "Модерн на Патриарших",

GET /api/v1/partners/stats
```
