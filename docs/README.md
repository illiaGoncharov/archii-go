# Archi-Go

Мобильное приложение для сбора архитектурных элементов. Гуляй по городу, фотографируй метопы, сандрики, капители — собирай коллекцию и создавай свои фасады.

## Технологии

- **Expo** (React Native) с TypeScript
- **Expo Router** для навигации
- **Zustand** для управления состоянием
- **Firebase** (Auth, Firestore, Storage)
- **react-native-maps** для карты
- **expo-camera** для съёмки

## Запуск

1. Установи зависимости:
```bash
npm install
```

2. Настрой Firebase:
   - Создай проект в Firebase Console
   - Добавь конфигурацию в `firebase-config.ts`

3. Запусти:
```bash
npm start
```

4. Тестируй на iPhone через **Expo Go**:
   - Скачай Expo Go из App Store
   - Отсканируй QR-код

## Структура

- `app/` — экраны (Expo Router)
- `components/` — переиспользуемые компоненты
- `services/` — Firebase и API
- `stores/` — Zustand stores
- `types/` — TypeScript типы
- `utils/` — вспомогательные функции

## MVP функции

- ✅ Карта с точками интереса (POI)
- ✅ Камера для съёмки элементов
- ✅ Коллекция найденных элементов
- ✅ Конструктор фасадов
- ✅ Профиль с статистикой
- ✅ Геймификация (очки, редкость, уровни)
