# Archi-Go: Монетизация и развитие в контексте российской экономики

> **Промпт для разработки стратегии**

---

## Контекст проекта

**Archi-Go** — мобильное приложение для сбора архитектурных элементов. Геймифицированный способ изучения городской архитектуры: пользователи гуляют по городу, фотографируют архитектурные детали (метопы, сандрики, капители, маскароны), собирают коллекцию, получают очки и создают виртуальные фасады.

### Текущий статус
- MVP реализован (7 экранов, 42 файла)
- Стек: React Native (Expo), TypeScript, Firebase, Zustand
- 15 архитектурных элементов, 5 POI в Москве
- Геймификация: уровни, очки, редкость элементов, конструктор фасадов
- Дизайн-философия: "Структурная эстетика" — минимализм, уважение к архитектуре

### Уникальность продукта
1. **Образовательный геймплей** — изучение архитектуры через игру
2. **Коллекционирование** — психология "собирательства" работает
3. **Креативный инструмент** — конструктор фасадов
4. **Городское исследование** — мотивирует гулять и замечать детали
5. **Культурная миссия** — сохранение и популяризация архитектурного наследия

## Вопросы для проработки

### 1. Модель монетизации для России

**Базовые вопросы:**
- Какая цена подписки оптимальна для российского рынка? (Учитывая средний чек подписок: Яндекс.Плюс 299₽, VK Музыка 169₽, Netflix ~999₽)
- Freemium vs Подписка vs Единоразовая покупка — что работает лучше для образовательно-культурных приложений?
- Как обойти ограничения App Store для российских карт? (RuStore, Huawei AppGallery, прямые платежи)
- Какие in-app purchases будут востребованы? (Наборы элементов, маршруты, эксклюзивный контент)

**Государственное финансирование:**
- Какие гранты доступны? (Президентский фонд культурных инициатив, Фонд Потанина, гранты Минкульта)
- Как позиционировать проект для грантовой заявки? (Сохранение культурного наследия, образование, туризм)
- Возможно ли финансирование по линии "Развитие внутреннего туризма"?
- Региональные программы поддержки креативных индустрий (Москва, СПб, Казань)

### 2. B2G — Взаимодействие с государством

**Министерство культуры и регионы:**
- Как предложить Archi-Go как инструмент популяризации культурного наследия?
- Возможна ли интеграция с реестром объектов культурного наследия (ОКН)?
- Партнёрство с программой "Пушкинская карта" — для молодёжной аудитории
- Интеграция в туристические программы регионов ("Золотое кольцо", "Серебряное ожерелье")

**Городские администрации:**
- Москомархитектура: интеграция в архитектурные фестивали (День архитектуры, Арх Москва)
- КГИОП СПб: контент по охраняемым зданиям
- Казань: партнёрство с программами сохранения исторического центра

**Образование:**
- Включение в школьные/вузовские программы по краеведению и МХК
- Создание методических материалов для учителей
- Партнёрство с детскими лагерями и образовательными центрами

### 3. B2B — Культурные институции

**Музеи (Гараж, Щусев, Эрмитаж):**
- White label решение: брендированные маршруты от музеев
- Совместные события: "Охота за элементами" на выходных
- Эксклюзивный контент: элементы, доступные только при посещении музея
- Revenue share: процент от подписок пользователей, привлечённых музеем

**Культурные центры (ДК Лурье, GES-2, Стрелка):**
- Мероприятия: квесты, лекции, мастер-классы с использованием приложения
- Резиденции: создание контента с приглашёнными архитекторами/историками
- Коллаборации: совместные маршруты, временные выставки в приложении

**Архитектурные школы (МАРШ, МАРХИ):**
- Образовательные модули: курсы с интеграцией практики в городе
- Студенческие проекты: наполнение базы элементов силами студентов
- Исследования: данные о городских прогулках для урбанистических исследований

**Экскурсионные бюро и гиды:**
- Инструмент для гидов: создание авторских маршрутов
- Монетизация: продажа маршрутов через приложение (revenue share)
- Сертификация: "Официальный гид Archi-Go"

### 4. B2C — Пользователи

**Сегментация:**
- Энтузиасты архитектуры (основная ЦА, 25-45 лет, городские жители)
- Туристы (внутренний туризм растёт)
- Студенты и школьники (образовательный фокус)
- Коллекционеры и геймеры (мотивация "собрать всё")
- Фотографы и блогеры (контент для соцсетей)

**Ценностное предложение:**
- Бесплатно: базовая коллекция, карта, камера, простой конструктор
- Premium (199-299₽/мес): полная коллекция, авторские маршруты, экспорт фасадов
- Pro (499-799₽/мес): создание своих маршрутов, аналитика, коммерческое использование

### 5. Локальное сообщество

**Краеведы и историки:**
- Роль "Хранителей": верификация находок, написание описаний
- Авторские маршруты от известных краеведов
- Партнёрство: Архнадзор, Живой город, Moscow Heritage

**Городские активисты:**
- Интеграция с проектами по сохранению наследия
- Документирование утраченных элементов (раздел "Память")
- Crowdsourcing: пользователи сообщают о разрушающихся элементах

**Блогеры и инфлюенсеры:**
- Партнёрская программа: промокоды, реферальные бонусы
- Контент-интеграции: обзоры, прогулки, челленджи
- UGC: пользовательские фасады в соцсетях
