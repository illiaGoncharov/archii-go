## Цели и задачи

### Зачем нужны мероприятия

```
1. ПРИВЛЕЧЕНИЕ ПОЛЬЗОВАТЕЛЕЙ
   • Новые скачивания приложения
   • Конверсия участников в пользователей
   • Сарафанное радио

2. ВОВЛЕЧЕНИЕ СООБЩЕСТВА
   • Активация неактивных пользователей
   • Укрепление связи с продуктом
   • Создание комьюнити

3. ПАРТНЁРСТВА
   • Совместные проекты с музеями/центрами
   • Демонстрация ценности партнёрам
   • PR и медийное покрытие

4. КОНТЕНТ
   • Фото и видео для соцсетей
   • UGC от участников
   • Истории успеха

5. МОНЕТИЗАЦИЯ
   • Платные мероприятия
   • Продажа подписок участникам
   • Привлечение спонсоров
```

## Форматы мероприятий

### 1. Архитектурные квесты

```
ФОРМАТ: «ОХОТА ЗА ЭЛЕМЕНТАМИ»

Описание:
Командное соревнование — кто найдёт больше элементов 
за ограниченное время (2 часа).

Структура:
• Сбор участников (15 мин)
• Брифинг и правила (15 мин)
• Квест по району (90 мин)
• Подведение итогов, награждение (30 мин)

Параметры:
• Участники: 20-50 человек
• Команды: 3-5 человек
• Продолжительность: 2.5 часа
• Цена: 500-800 ₽
• Частота: 2 раза в месяц

Призы:
• 1 место: Годовая подписка + мерч
• 2 место: 6 месяцев подписки
• 3 место: 3 месяца подписки
• Все: скидка 30% на подписку

Что нужно:
• Организатор (1 человек)
• Волонтёры на контрольных точках (2-3)
• Место для старта/финиша (кафе, парк)
• Призы
```

### 2. Архитектурные прогулки

```
ФОРМАТ: «ПРОГУЛКА С ЭКСПЕРТОМ»

Описание:
Экскурсия по маршруту из приложения с комментариями 
гида или краеведа. Участники используют приложение 
для сбора элементов.

Структура:
• Знакомство и введение (15 мин)
• Прогулка с остановками (2 часа)
• Свободное время для фото (15 мин)
• Wrap-up и вопросы (15 мин)

Параметры:
• Участники: 15-30 человек
• Продолжительность: 2.5 часа
• Цена: 800-1500 ₽
• Частота: 2-4 раза в месяц

Варианты:
• Тематические: «Модерн», «Классицизм»
• Районные: «Арбат», «Патриаршие»
• Авторские: от известных краеведов
• Ночные: летом, с подсветкой

Что нужно:
• Гид/эксперт
• Организатор
• Рация или Telegram-чат для связи
```

### 3. Мастер-классы

```
ФОРМАТ: «СОЗДАЙ СВОЙ ФАСАД»

Описание:
Indoor-мероприятие, где участники учатся создавать 
фасады в конструкторе приложения + базовые знания 
об архитектурных стилях.

Структура:
• Лекция: архитектурные стили (30 мин)
• Демонстрация конструктора (15 мин)
• Практика: создание фасада (45 мин)
• Показ работ, обсуждение (30 мин)

Параметры:
• Участники: 15-25 человек
• Продолжительность: 2 часа
• Цена: 600-1000 ₽
• Частота: 1-2 раза в месяц

Варианты:
• Для детей (с родителями)
• Корпоративный формат
• Онлайн-версия (Zoom)

Что нужно:
• Помещение с проектором
• Лектор/ведущий
• Wi-Fi
• Раздаточные материалы
```

### 4. Лекции

```
ФОРМАТ: «АРХИТЕКТУРНЫЙ ЛИКБЕЗ»

Описание:
Образовательная лекция от эксперта о конкретном 
стиле, периоде или архитекторе.

Структура:
• Лекция (60-90 мин)
• Q&A (30 мин)
• Нетворкинг (по желанию)

Параметры:
• Участники: 30-100 человек
• Продолжительность: 1.5-2 часа
• Цена: 400-800 ₽
• Частота: 1-2 раза в месяц

Темы:
• «Модерн: русская версия»
• «Конструктивизм: как его понимать»
• «Классицизм в Москве: главные здания»
• «Архитектор Шехтель: жизнь и работы»
• «Утраченная Москва: что мы потеряли»

Спикеры:
• Историки архитектуры
• Краеведы
• Сотрудники музеев
• Преподаватели МАРХИ/МАРШ
```

### 5. Сезонные события

```
ФОРМАТ: «ДЕНЬ АРХИТЕКТУРЫ»

Описание:
Большое ежегодное мероприятие, приуроченное к 
Всемирному дню архитектуры (первый понедельник октября).

Программа:
• Утро: квест по центру города
• День: лекции и мастер-классы (площадка партнёра)
• Вечер: вечеринка / нетворкинг

Параметры:
• Участники: 100-300 человек
• Продолжительность: целый день
• Цена: 1500-3000 ₽
• Частота: 1 раз в год

Партнёры:
• Площадка: GES-2, Стрелка, Гараж
• Спонсоры: архитектурные бюро, застройщики
• Медиапартнёры: Strelka Mag, The Village
```

### 6. Корпоративные мероприятия

```
ФОРМАТ: «ТИМБИЛДИНГ С ARCHI-GO»

Описание:
Кастомизированный квест или прогулка для 
корпоративных команд.

Варианты:
• Квест-соревнование между отделами
• Прогулка + лекция
• Командное создание фасадов
• Квест + обед/ужин

Параметры:
• Участники: 15-50 человек
• Продолжительность: 2-4 часа
• Цена: 30,000-100,000 ₽ (за группу)
• Включено: организация, гид, призы, кейтеринг

Целевые клиенты:
• IT-компании
• Архитектурные бюро
• Девелоперы
• Культурные организации
• HR-отделы крупных компаний
```

## Партнёрские мероприятия

### Модели партнёрства

```
МОДЕЛЬ A: ПЛОЩАДКА

Партнёр предоставляет площадку, мы — контент и организацию.

Пример: Лекция в Гараже
• Гараж: зал, продвижение по своим каналам
• Archi-Go: лектор, контент, регистрация, продажи

Распределение:
• Доход: 100% Archi-Go
• Расходы: 100% Archi-Go
• Партнёр получает: трафик, контент, PR


МОДЕЛЬ B: СОВМЕСТНОЕ СОБЫТИЕ

50/50 — совместная организация и разделение дохода.

Пример: Квест от Музея Щусева
• Музей: экспертиза, бренд, часть промо
• Archi-Go: организация, приложение, часть промо

Распределение:
• Доход: 50/50
• Расходы: 50/50


МОДЕЛЬ C: БРЕНДИРОВАННОЕ СОБЫТИЕ

Партнёр заказывает событие под свой бренд.

Пример: Тимбилдинг для компании X
• Компания: бюджет, участники
• Archi-Go: полная организация

Распределение:
• Фиксированная цена: 50,000-150,000 ₽
```

## Операционная модель

### Команда на событие

```
МАЛОЕ СОБЫТИЕ (20-30 человек):
• Организатор / ведущий — 1 человек
• Гид (если прогулка) — 1 человек
• Итого: 1-2 человека

СРЕДНЕЕ СОБЫТИЕ (50-100 человек):
• Организатор — 1 человек
• Ведущий / лектор — 1 человек
• Волонтёры — 2-3 человека
• Итого: 4-5 человек

БОЛЬШОЕ СОБЫТИЕ (100+ человек):
• Продюсер — 1 человек
• Организаторы — 2-3 человека
• Ведущие / гиды — 3-5 человек
• Волонтёры — 5-10 человек
• Итого: 10-20 человек
```

### Чек-лист подготовки события

```
ЗА 4 НЕДЕЛИ:
□ Определить формат и дату
□ Забронировать площадку
□ Договориться с гидом/лектором
□ Создать страницу события (Timepad)

ЗА 2 НЕДЕЛИ:
□ Запустить продажи/регистрацию
□ Анонс в соцсетях
□ Рассылка по базе
□ Промо у партнёров

ЗА 1 НЕДЕЛЮ:
□ Напоминание зарегистрированным
□ Подготовить материалы
□ Брифинг команды
□ Проверить площадку

В ДЕНЬ СОБЫТИЯ:
□ Проверка готовности
□ Регистрация участников
□ Фото и видео
□ Проведение
□ Сбор обратной связи

ПОСЛЕ СОБЫТИЯ:
□ Отправить фото участникам
□ Пост в соцсетях
□ Благодарность партнёрам
□ Анализ метрик
□ Отчёт
```

## Метрики успеха

### Метрики события

| Метрика | Цель | Как измерять |
|---------|------|--------------|
| Заполняемость | > 80% | Билеты / мест |
| NPS | > 60 | Опрос после |
| Конверсия в скачивание | > 70% | Опрос + аналитика |
| Конверсия в подписку | > 10% | Промокоды |
| UGC | 20+ постов | Мониторинг хештега |
| Repeat rate | > 30% | Повторные участники |

### Опрос после события

```
1. Как вы оцениваете событие? (1-10)

2. Что понравилось больше всего?
   □ Контент / информация
   □ Организация
   □ Атмосфера
   □ Приложение
   □ Ведущий / гид
   □ Другое: ___

3. Что можно улучшить?
   [Открытый ответ]

4. Скачали ли вы приложение?
   □ Да, на событии
   □ Было до события
   □ Нет
```
