# Точки интереса (POI) для Москвы и Санкт-Петербурга

## Описание

Коллекция из 27 детально описанных архитектурных объектов:
- **15 точек в Москве**
- **12 точек в Санкт-Петербурге**

Каждая точка содержит подробную информацию об истории здания, архитектурном стиле, элементах для поиска и интересные факты.

## Структура данных

```typescript
interface DetailedPOI {
  id: string;                    // Уникальный идентификатор (msk_1, spb_1, etc.)
  name: string;                  // Краткое название объекта
  address: string;               // Точный адрес
  city: string;                  // "Москва" или "Санкт-Петербург"
  descriptionShort: string;      // Краткое описание (1-2 предложения)
  descriptionFull: string;       // Полное описание (2 абзаца)
  style: string;                 // Архитектурный стиль
  elements: string[];            // Список архитектурных элементов для поиска
  funFact: string;              // Интересный факт
  coordinates: {
    latitude: number;
    longitude: number;
  };
  difficulty: 'easy' | 'medium' | 'hard';  // Сложность поиска элементов
}
```

## Использование

### Импорт

```typescript
import { 
  moscowSpbPOIs, 
  getMoscowPOIs, 
  getSpbPOIs,
  getPOIById,
  getPOIsByStyle,
  getPOIsByDifficulty
} from '@/constants/poi-moscow-spb';
```

### Примеры

```typescript
// Получить все точки
const allPOIs = moscowSpbPOIs;

// Получить только московские точки
const moscowPOIs = getMoscowPOIs();

// Получить только петербургские точки
const spbPOIs = getSpbPOIs();

// Получить точку по ID
const englishClub = getPOIById('msk_1');

// Получить все точки в стиле модерн
const modernPOIs = getPOIsByStyle('Модерн');

// Получить легкие точки для новичков
const easyPOIs = getPOIsByDifficulty('easy');
```

## Уровни сложности

### Easy (10 точек)
Хорошо видимые здания с легкодоступными элементами. Подходят для новичков.

### Medium (11 точек)
Требуют внимательного осмотра фасада. Некоторые элементы на высоте.

### Hard (6 точек)
Редкие или труднодоступные элементы. Для опытных охотников за архитектурой.

## Интеграция с приложением

### В карте
Используйте координаты для отображения POI на карте:

```typescript
import MapView, { Marker } from 'react-native-maps';
import { moscowSpbPOIs } from '@/constants/poi-moscow-spb';

<MapView>
  {moscowSpbPOIs.map(poi => (
    <Marker
      key={poi.id}
      coordinate={{
        latitude: poi.coordinates.latitude,
        longitude: poi.coordinates.longitude
      }}
      title={poi.name}
      description={poi.descriptionShort}
    />
  ))}
</MapView>
```

### В модальном окне
Показывайте детальную информацию при выборе точки:

```typescript
const [selectedPOI, setSelectedPOI] = useState<DetailedPOI | null>(null);

<Modal visible={!!selectedPOI}>
  <Text>{selectedPOI?.name}</Text>
  <Text>{selectedPOI?.address}</Text>
  <Text>{selectedPOI?.descriptionFull}</Text>
  <Text>Стиль: {selectedPOI?.style}</Text>
  <Text>Ищи: {selectedPOI?.elements.join(', ')}</Text>
  <Text>Интересный факт: {selectedPOI?.funFact}</Text>
</Modal>
```

### В квестах
Используйте список элементов для создания заданий:

```typescript
const quest = {
  poiId: 'msk_12',
  elements: getPOIById('msk_12')?.elements,
  difficulty: getPOIById('msk_12')?.difficulty
};
```

## Дополнительные возможности

### Фильтрация по городу

```typescript
// Показать только московские точки на карте
const cityFilter = 'Москва';
const filteredPOIs = moscowSpbPOIs.filter(poi => poi.city === cityFilter);
```

### Группировка по стилю

```typescript
const groupedByStyle = moscowSpbPOIs.reduce((acc, poi) => {
  if (!acc[poi.style]) acc[poi.style] = [];
  acc[poi.style].push(poi);
  return acc;
}, {} as Record<string, DetailedPOI[]>);
```

### Сортировка по сложности

```typescript
const sortedByDifficulty = [...moscowSpbPOIs].sort((a, b) => {
  const order = { easy: 1, medium: 2, hard: 3 };
  return order[a.difficulty] - order[b.difficulty];
});
```
