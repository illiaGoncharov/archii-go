### Структура заявки ПФКИ

#### 1. Название проекта

```
«Archi-Go: цифровая платформа для изучения и популяризации 
архитектурного наследия российских городов»
```

**Альтернативные варианты:**
- «Архитектурная азбука: мобильное приложение для молодёжи»
- «Собери свой город: геймификация культурного наследия»

#### 2. Краткое описание (до 1000 знаков)

```
Проект «Archi-Go» — мобильное приложение, которое превращает изучение 
архитектурного наследия в увлекательную игру.

Пользователи гуляют по городу, фотографируют архитектурные элементы 
(метопы, капители, маскароны, наличники), собирают коллекцию и создают 
виртуальные фасады. Каждый элемент сопровождается образовательным 
контентом: история, стиль, значение в архитектуре.

Цель проекта — привлечь молодую аудиторию (18-35 лет) к изучению 
архитектуры и сформировать культуру бережного отношения к 
архитектурному наследию.

В рамках гранта планируется:
• Расширение базы элементов до 100+ позиций
• Создание 20 авторских маршрутов по Москве и Санкт-Петербургу
• Партнёрство с 5 музеями и культурными центрами
• Проведение 10 публичных мероприятий
• Охват 15 000+ пользователей

Проект реализуется в партнёрстве с Музеем архитектуры им. Щусева и 
Институтом «Стрелка» (планируемые партнёры).
```

#### 3. Полное описание проекта

##### 3.1 Актуальность

```
ПРОБЛЕМА:

Архитектурное наследие российских городов недостаточно изучено и 
популяризировано среди молодого поколения. По данным ВЦИОМ (2023), 
лишь 23% россиян в возрасте 18-35 лет интересуются историей 
архитектуры своего города.

Существующие форматы (экскурсии, книги, лекции) не отвечают привычкам 
цифрового поколения. При этом 78% молодых россиян ежедневно используют 
смартфон для развлечений и обучения.

Разрыв между культурным наследием и молодёжью ведёт к:
• Утрате интереса к сохранению памятников
• Отсутствию общественного контроля за состоянием зданий
• Потере культурной идентичности городов

РЕШЕНИЕ:

Archi-Go использует геймификацию — подход, доказавший эффективность в 
образовании. Исследования показывают, что геймифицированное обучение 
повышает вовлечённость на 60% и улучшает запоминание на 40% (Gartner, 2023).

Мобильный формат позволяет интегрировать обучение в повседневную жизнь: 
пользователь узнаёт об архитектуре во время прогулки, без специальных 
усилий и выделенного времени.
```
