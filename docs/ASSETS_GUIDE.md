# Гайд по созданию визуальных элементов

## Проблема

Нужно создать изображения архитектурных элементов и зданий для приложения. Требования:
- **Минимальные средства** (один человек, инди-разработка)
- **Художественное уважение** к деталям, пропорциям, формам
- **Единый стиль** визуализации
- **Быстрое создание** новых элементов

## Решение: Стилизованная векторная графика

### Концепция стиля

**Line Art** (линейная графика) + **Минимальные тени**

Почему этот стиль:
- ✅ Не требует профессиональных навыков 3D или рисования
- ✅ Фокус на формах и пропорциях (как в архитектуре)
- ✅ Легко масштабируется
- ✅ Единый стиль для всех элементов
- ✅ Быстрое создание новых элементов

### Референсы стиля

Вдохновение:
- **Monument Valley** (игра) — геометрические формы, чистые линии
- **Architectuul** — минималистичные иллюстрации архитектуры
- **Isometric architectural drawings** — изометрия, но упрощённая

## Инструменты

### Вариант 1: Figma (рекомендуется)

**Почему Figma:**
- Бесплатная
- В браузере или десктоп
- Векторная графика
- Простой интерфейс
- Экспорт в PNG/SVG

**Как начать:**
1. Создай файл Figma
2. Установи артборды 300x300px
3. Используй инструмент Pen (P) для контуров
4. Rectangle (R) для базовых форм
5. Экспортируй в PNG 2x (600x600px)

### Вариант 2: Adobe Illustrator

Если есть доступ — отличный инструмент для векторной графики.

### Вариант 3: Inkscape

Бесплатная open-source альтернатива Illustrator.

### Вариант 4: iPad + Procreate

Если есть iPad — можно рисовать стилусом, экспортировать PNG.

## Процесс создания элемента

### Шаг 1: Референсы

1. **Найди фото элемента** (Google Images, Wikimedia)
2. **Изучи пропорции** — обрати внимание на:
   - Соотношение частей
   - Основную форму
   - Ключевые детали

### Шаг 2: Упрощение

Архитектурный элемент → Геометрическая абстракция

**Пример: Капитель**
```
Реальная капитель:
- Сложные волюты
- Детализированные листья
- Тени и текстуры

Упрощённая капитель:
- Простая S-образная форма для волют
- 3-5 листьев базовой формы
- Чистые линии
- Один цвет с лёгкой тенью
```

### Шаг 3: Создание в Figma

#### Базовый шаблон

1. **Артборд**: 300x300px
2. **Фон**: белый или прозрачный
3. **Цвет линий**: #111827 (тёмно-серый)
4. **Толщина линий**: 2-3px
5. **Цвет заливки**: #F3F4F6 (светло-серый) или #FFFFFF

#### Техника рисования

```
1. Начни с базовых форм (прямоугольники, круги, эллипсы)
2. Используй Pen tool для сложных контуров
3. Boolean operations для соединения фигур
4. Добавь лёгкую тень (Gradient):
   - От #00000010 до #00000000
   - Внутри формы снизу
5. Экспортируй PNG 2x (600x600px)
```

### Шаг 4: Стандартизация

#### Все элементы должны:

- **Размер**: 300x300px (экспорт 600x600px)
- **Центрирование**: элемент в центре артборда
- **Padding**: отступ 20-30px от краёв
- **Цвета**: из палитры (см. DESIGN_GUIDE.md)
- **Линии**: одинаковая толщина (2-3px)

#### Стили по редкости (опционально)

- **Common**: монохром, серые тона
- **Rare**: добавить синий акцент
- **Epic**: добавить фиолетовый акцент
- **Legendary**: добавить золотой акцент

## Примеры элементов

### 1. Метопа (простой)

```
┌─────────────────┐
│                 │
│  ┌───────────┐  │
│  │           │  │
│  │  Рельеф   │  │  ← Прямоугольник с лёгкой тенью
│  │           │  │
│  └───────────┘  │
│                 │
└─────────────────┘
```

**В Figma:**
1. Rectangle 260x200px
2. Border 2px, #111827
3. Fill #F3F4F6
4. Внутри — простой геометрический узор (круги, линии)

### 2. Сандрик (средний)

```
        ▲
       ╱ ╲
      ╱   ╲
     ╱─────╲
    │       │
    └───────┘
```

**В Figma:**
1. Треугольник (Polygon, 3 sides)
2. Прямоугольник снизу
3. Union (Boolean)
4. Добавить лёгкую тень

### 3. Капитель (сложный)

```
    ╭─╮ ╭─╮
    │ ╰─╯ │
    ╰─────╯
    │     │
```

**В Figma:**
1. Pen tool для волют
2. Несколько кривых линий
3. Прямоугольник для базы
4. Группировка

## Альтернативный подход: Фотографии + обработка

Если не хочешь рисовать:

### Вариант A: Фото + фильтры

1. Сфотографируй элемент
2. Обрежь в квадрат
3. Конвертируй в ЧБ
4. Увеличь контраст
5. Примени фильтр "Poster Edges" (Photoshop) или аналог

### Вариант B: Трассировка фото

1. Загрузи фото в Figma/Illustrator
2. Обведи контуры вручную поверх фото
3. Удали фото, оставь контуры
4. Упрости линии

### Вариант C: AI генерация

**Midjourney / DALL-E / Stable Diffusion:**

Промпт:
```
"architectural element [название], line art style, minimalist, black and white, 
isometric view, clean lines, vector illustration, white background, 
architectural drawing style"
```

Примеры:
- "corinthian capital, line art style, minimalist..."
- "baroque cartouche, line art style, minimalist..."

**Плюсы**: быстро, профессионально  
**Минусы**: нужна подписка, не всегда точные пропорции

## Примерное время

- **Простой элемент** (метопа, руст): 15-30 минут
- **Средний элемент** (сандрик, капитель): 30-60 минут
- **Сложный элемент** (маскарон, атлант): 1-2 часа
- **Освоение Figma**: 2-3 часа практики

## Итого

**Рекомендуемый подход:**

1. Используй Figma (бесплатно, просто)
2. Стиль: line art, минимализм
3. Опирайся на референсы, но упрощай
4. Начни с 5-10 простых элементов
5. Постепенно усложняй

**Результат:**
- Единый стиль
- Художественное уважение к формам
- Быстрое создание новых элементов
- Масштабируемость

---

**Пример готового workflow:**

1. Открыл Figma → создал артборд 300x300px

4. Упростил детали

6. Экспорт PNG 600x600px

8. Обновил imageUrl в Firestore
9. **Время**: 45 минут
